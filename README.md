# NextJS-Caching-Practice

NextJS Caching System을 이해하고 Caching을 재현하기 위한 코드입니다.

Next.js에서 Navigation 동안 적용되는 Route Cache

Next.js의 Fetch API에 내장되어 있는 Request Memoization, Data Cache를 이해 할 수 있습니다.

# Getting Start

```
npm i
npm run build
npm run start
```
